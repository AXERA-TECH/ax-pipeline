# ax-pipeline
基于 AXear-Pi (AX620A) 的 Pipeline 示例，展示 ISP、图像处理、NPU、编码、显示 模块基础调用方法，利于用户二次开发自己的多媒体应用。

## 已支持开发板

- AXera-Pi(AX620A)
